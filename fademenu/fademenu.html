A class to turn an appropriately styled unordered list (<ul>) into a (potentially multi-tiered) fading javascript menu.

Arguments

  • elem

    The list element (any collection of list items) from which to create the menu.

  • options

    An optional object containing the following properties:

    Property nameDescriptionDefault value
    fadeRecursiveSpecifies whether to apply FadeMenu to nested lists too.true
    fadeInSpeedThe time in milliseconds for each menu item to fade in150
    fadeOutSpeedThe time in milliseconds for each menu item to fade out100
    ignoreClassYou can tell FadeMenu to ignore <ul>s with a specific classfalse
    fadeOutDelayThe time in milliseconds after the mouse leaves the menu before it will start to fade out100

Examples

JavaScript

new FadeMenu('fademenu',{recursive: true, fadeOutDelay: 200});
